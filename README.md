
# ✨ Note Cleaner AI

A smart web app that converts messy, unstructured notes into clean, readable formats using rule-based NLP techniques.

## 🚀 Features

- 📝 Converts messy notes into structured bullet points
- 📚 Exam Mode → Extracts keywords for quick revision
- ✨ Fun Mode → Stylized, engaging output
- 📒 Notebook-style UI for a natural note-taking experience
- 🎨 Theme toggle (Default / Dark / Pastel)
- ⚡ Fast and lightweight (runs locally with Flask)

## 🧠 How It Works

The system processes raw text by:
- Cleaning unnecessary spaces
- Splitting text into meaningful chunks
- Extracting key terms
- Formatting output based on selected mode

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Concepts: Text Processing, Basic NLP, UI/UX Design

## 📸 Demo

Paste messy notes → Click "Clean Notes" → Get structured output instantly.

## 📂 Project Structure
note-cleaner-ai/
│
├── frontend/
│ ├── index.html
│ ├── style.css
│ └── script.js
│
├── backend/
│ └── app.py


## ▶️ How to Run

1. Clone the repository

2. Install dependencies:
    pip install flask flask-cors

3. Run backend:
    python app.py

4. Open `index.html` in browser (or Live Server)

## 💡 Future Improvements

- AI-based summarization
- Topic detection
- Keyword highlighting
- Export as PDF
